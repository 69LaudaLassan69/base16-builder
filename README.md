# Base16 Builder
Build color variations of Base16 with YAML scheme definitions and ERB templates. 

## Usage
    > base16
Build all schemes

    > base16 default.yml
Build only the "default" theme

## Templates
* HTML Preview
* Vim
* TextMate
* iTerm 2
